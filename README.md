# Workflow – Frontend

A clean, modern, and responsive todo list web app that helps you organize your life.

### Features

- Beautiful dashboard with task statistics
- Full task management (add, edit, delete, complete)
- Smart filtering & search across all tasks
- Separate views: All Tasks • To Do • In Progress • Completed
- Fully responsive (mobile + desktop) with mobile bottom navigation
- Smooth auth flow (login / register)
- Persistent session (stay logged in)
- Loading states & error handling
- Minimal, focused, and fast

### Tech Stack

- React + Vite
- Context API + useReducer (global state)
- React Router v6 (nested routes)
- Pure CSS modules (no framework)
- Mobile-first design

> Works out of the box with the Workflow backend

Built to feel simple — because getting things done should be.
